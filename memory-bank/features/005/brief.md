# 05_forum_structure

## Brief
[GitHub Issue](https://github.com/Pegorino82/lfcru_forum/issues/5)

## Проблема

Форум существует как идея, но не реализован как функциональная единица. Пользователь не может зайти на форум, увидеть разделы и темы, прочитать или оставить сообщение — структура «разделы → темы → сообщения» отсутствует полностью.

## Для кого

- Гости сайта (основной трафик, не авторизованы) — читают открытые разделы, темы и посты
- Зарегистрированные пользователи — оставляют посты
- Админы, модераторы - могут создавать темы, разделы

## Желаемый результат

Пользователь заходит на форум, видит список разделов (sections). Переходя в раздел, видит список тем (topics). Открывая тему, читает посты (posts). Зарегистрированный пользователь может написать пост или ответить (прокомментировать, процитировать) на пост другого пользователя.

## Acceptance Criteria

**Чтение (Гость, все роли)**
- [ ] Страница `/forum` отображает список разделов (название, описание, счётчик тем)
- [ ] Страница раздела `/forum/sections/:id` отображает список тем (название, автор, дата последнего поста, счётчик постов)
- [ ] Страница темы `/forum/topics/:id` отображает список постов с текстом, именем автора и датой
- [ ] С главной страницы (блок «Активность форума») переходы ведут на соответствующие страницы форума

**Создание контента (Зарегистрированный пользователь)**
- [ ] Авторизованный пользователь может создать пост в теме
- [ ] Авторизованный пользователь может ответить на пост другого пользователя — ответ отображается с цитатой (аналогично feature 004)
- [ ] Неавторизованный пользователь не может создать пост и не видит кнопку «Ответить»

**Управление структурой (Модератор, Администратор)**
- [ ] Модератор/Админ может создать раздел (название, описание)
- [ ] Модератор/Админ может создать тему в разделе (название, раздел)
- [ ] Пользователь без прав не может создать раздел или тему

## Контекст

Главная страница уже показывает блок «Активность форума», но переходы с него никуда не ведут — роутов и страниц форума нет. Это базовая функциональность сайта, без которой форум не существует как таковой.
