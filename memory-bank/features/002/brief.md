# 02_layout_base

## Brief
**GitHub Issue Title:** UI: базовый layout сайта

**GitHub Issue Description:**
Создать базовый layout для всех страниц сайта.

Функциональность:
- Header (логотип, навигация)
- Footer
- Базовый HTML-шаблон
- Поддержка partials

Требования:
- Использовать html/template
- Подготовить layout для расширения

Acceptance Criteria:
- Все страницы используют единый layout
- Header и footer отображаются
- Partial'ы работают
