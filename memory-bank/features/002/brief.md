# 02_layout_base

## Brief

[GitHub Issue](https://github.com/Pegorino82/lfcru_forum/issues/2)

## Проблема

На разных страницах сайта отсутствуют хэдер или футер: пользователь не понимает, что находится на одном ресурсе, и теряет навигацию

## Для кого

Для всех посетителей сайта.

## Желаемый результат

При переходе между разделами сайта и форума пользователь видит одинаковые хэдер и футер на каждой странице.
Хэдер содержит логотип и навигацию, футер — ссылки и копирайт; оба присутствуют на каждой странице сайта и форума.

## Acceptance Criteria

 - Хэдер отображается на всех страницах сайта и форума     
 - Футер отображается на всех страницах сайта и форума                       
 - При переходе между разделами хэдер и футер не исчезают и не меняют вид
