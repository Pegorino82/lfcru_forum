# 03_homepage

## Brief
[GitHub Issue](https://github.com/Pegorino82/lfcru_forum/issues/3)

## Проблема

Сайт не имеет главной страницы — пользователь, заходя на `/`, не видит
никакого контента. Это первый экран, который формирует впечатление о
ресурсе и определяет, останется ли посетитель.

## Для кого

- Гости сайта (основной трафик, не авторизованы)
- Зарегистрированные пользователи

## Желаемый результат

Пользователь, заходя на `/`, сразу видит актуальное состояние сайта:
свежие новости клуба, информацию о ближайшем матче и информацию о последних
сообщениях форума — без необходимости переходить в другие разделы.

## Acceptance Criteria

- [ ] На странице `/` отображается список из 5 последних новостей
      (заголовок + дата публикации), отсортированных по дате публикации (новые первыми); если новостей нет —
      блок показывает текст «На сайте еще не добавлены новости»
- [ ] На странице отображается блок ближайшего матча
      (соперник, дата, турнир); если матч не запланирован —
      блок показывает текст «Ближайших матчей нет»
- [ ] На странице отображается список из 5 тем с последней активностью (название темы, автор последнего сообщения, дата последнего сообщения), отсортированных по дате последнего сообщения (новые первыми); если сообщений нет —
      блок показывает текст «В форуме пока нет тем»
- [ ] Все три блока видны без прокрутки на экране ≥ 1280×900px
- [ ] Страница доступна для гостей без авторизации

## Контекст

Проект находится на этапе начальной разработки. Отсутствие главной
страницы блокирует нормальную навигацию и не позволяет развивать UI
остальных разделов. Это первый публичный экран сайта.
