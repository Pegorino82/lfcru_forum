# FT-003: Homepage

## What

### Problem

Сайт не имеет главной страницы — при заходе на `/` пользователь не видит никакого контента. Это первый экран, который формирует впечатление о ресурсе и определяет, останется ли посетитель.

Отсутствие главной страницы блокирует нормальную навигацию и развитие UI остальных разделов.

Общий контекст: [`../../domain/problem.md`](../../domain/problem.md).

### Outcome

| Metric ID | Metric | Baseline | Target | Measurement method |
|---|---|---|---|---|
| `MET-01` | Гость, открывший главную, видит ≥1 новость | 0% | 100% при наличии контента | E2E-тест на `/` |

### Scope

- `REQ-1` Страница `/` отображает 5 последних новостей (заголовок + дата публикации, новейшие первыми); при отсутствии — «На сайте еще не добавлены новости»
- `REQ-2` Страница отображает блок ближайшего матча (соперник, дата, турнир); при отсутствии — «Ближайших матчей нет»
- `REQ-3` Страница отображает 5 тем с последней активностью (название, автор последнего поста, дата); при отсутствии — «В форуме пока нет тем»
- `REQ-4` Страница доступна для неавторизованных гостей

### Non-Scope

- `NS-1` Пагинация или «показать ещё» для блоков новостей и тем
- `NS-2` Персонализация блоков для авторизованных пользователей
- `NS-3` Автоматическое обновление блоков через SSE (WF-04 — отдельная фича)

### Constraints / Assumptions

- `CON-1` Все три блока должны быть видны без прокрутки на экране ≥1280×900px (layout-ограничение)

## Verify

### Exit Criteria

- `EC-1` Все `REQ-*` покрыты passing `SC-*`
- `EC-2` `CON-1` подтверждён визуально или через E2E-тест при viewport 1280×900

### Acceptance Scenarios

- `SC-1` Гость открывает `/` — в блоке «Новости» видит 5 заголовков с датами, отсортированных по убыванию
- `SC-2` Гость открывает `/` при отсутствии новостей — видит текст «На сайте еще не добавлены новости»
- `SC-3` Гость открывает `/` — блок «Матч» показывает соперника, дату, турнир
- `SC-4` Гость открывает `/` при отсутствии запланированных матчей — видит «Ближайших матчей нет»
- `SC-5` Гость открывает `/` — в блоке «Форум» видит 5 тем с метаданными
- `SC-6` Гость открывает `/` при пустом форуме — видит «В форуме пока нет тем»
- `SC-7` Гость открывает `/` на экране 1280×900 — все три блока видны без вертикального scroll
